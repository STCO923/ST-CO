Table vehicules introuvable dans Supabase
Copie ce SQL dans Supabase → SQL Editor puis exécute :
create table vehicules ( id uuid primary key default gen_random_uuid(), immatriculation text not null, marque text default '', chauffeur_nom text default '', statut text default 'disponible', kilometrage integer default 0, notes text default '', ct_date date, ct_echeance date, assurance_echeance date, assurance_compagnie text default '', assurance_mensuel numeric default 0, type_propriete text default 'propre', tarif_location numeric default 0, typologie text default '', company_id uuid, created_at timestamp default now() ); alter table vehicules enable row level security; create policy "public_all" on vehicules for all using (true) with check (true);
Migration requise — colonne assurance_mensuel manquante
Ta table vehicules existe mais n'a pas encore la colonne d'assurance mensuelle. Exécute ce SQL dans Supabase → SQL Editor :
alter table vehicules add column if not exists assurance_mensuel numeric default 0;
Migration requise — colonnes type_propriete / tarif_location
Exécute ce SQL dans Supabase → SQL Editor :
alter table vehicules add column if not exists type_propriete text default 'propre'; alter table vehicules add column if not exists tarif_location numeric default 0;
Migration requise — colonne typologie
Exécute ce SQL dans Supabase → SQL Editor :
alter table vehicules add column if not exists typologie text default '';
